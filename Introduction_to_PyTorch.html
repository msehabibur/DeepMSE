Introduction to Tensor

In programming, tensors are typically used in libraries like PyTorch and TensorFlow to represent data that will be used in machine learning and deep learning models. These tensors can hold a wide range of data, including but not limited to:

  • 1D Tensor: Time series data, sound waveforms, or any sequential data.
  • 2D Tensor: Grayscale images (where each pixel is a number), text data (in the form of word embeddings), or any tabular data.
  • 3D Tensor: Color images (often with dimensions for height, width, and color channels like RGB).
  • 4D Tensor: A batch of color images, or video data (where time is an additional dimension).
  • 5D Tensor: A batch of video data or any data that requires five dimensions to represent.

Tensors are particularly well-suited to represent data in machine learning because they can be efficiently processed using GPU acceleration, which is critical for training complex neural networks on large datasets. The operations on tensors (like addition, multiplication, etc.) are optimized to be fast and memory efficient, making them a cornerstone of numerical computing and deep learning.

import torch
import torch.nn as nn
import torch.optim as optim
from skorch import NeuralNetRegressor
import numpy as np
import matplotlib.pyplot as plt

# Create a synthetic dataset for regression
np.random.seed(42)
X = np.random.rand(100, 1)
y = 2 * X + 1 + 0.1 * np.random.randn(100, 1)

# Convert X and y to PyTorch tensors
X = torch.tensor(X, dtype=torch.float32)
y = torch.tensor(y, dtype=torch.float32)

# Define a simple neural network model for regression
class SimpleRegressionNN(nn.Module):
    def __init__(self, input_dim):
        super(SimpleRegressionNN, self).__init__()
        self.fc = nn.Linear(input_dim, 1)

    def forward(self, x):
        x = self.fc(x)
        return x

input_dim = 1  # Since we have one feature

# Create a NeuralNetRegressor using skorch
net = NeuralNetRegressor(
    SimpleRegressionNN(input_dim),
    max_epochs=1000,
    lr=0.01,
    optimizer=optim.SGD,
    criterion=nn.MSELoss,
    device='cuda' if torch.cuda.is_available() else 'cpu',
)

# Train the model
